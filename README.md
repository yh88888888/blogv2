# 스프링부트 블로그 만들기 v2

```
> Get, Post 만 사용!
> ajax만 json으로
```

## 1. 유저(user_tb)

- 회원가입
- 로그인

## 2. 게시글(board_tb)

- 게시글 쓰기
- 게시글 목록
- 게시글 상세보기
- 삭제
- 수정

## 3. 게시글 고급 기능(board_tb)

- 게시글 목록 페이징
- 검색하기

## 4. 러브(love_tb)

- 게시글 좋아요 (ajax)
- 게시글 좋아요 취소 (ajax)

## 5. 댓글(reply_tb)

- 댓글 쓰기
- 댓글 삭제
- 댓글 목록보기

## 6. 그 외 기능

- 필터(Script 보안)
- 유효성 검사 자동화(Valid AOP)
- 인증 체크 자동화(Interceptor)
- 예외처리(Global Exception Handler)
- 사진 업로드(Multipart Form Data / Base64)
